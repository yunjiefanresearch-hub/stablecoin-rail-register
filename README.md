An **open, versioned, machine-readable** register of how jurisdictions regulate stablecoins —
mapped clause by clause across **fifteen dimensions** and **two doctrinal spines** (the
permitted-activity / yield boundary and securities classification), with a **corridor-level
interoperability layer**. Built from primary sources, citable by DOI.

**Focus:** US · Hong Kong · EU · UK · Singapore · Mainland China  ·  **Doctrinal anchor:** United States (CLARITY Act §404 / GENIUS Act)
**Forthcoming jurisdictions:** Brazil (v0.3) · Taiwan (v0.4)  ·  **Status:** v0.3.0 · **Cadence:** quarterly diffs + event patches · **License:** CC-BY-4.0 (data) / Apache-2.0 (code)

---

## What this is — and what it is not
This is **not** another broad, shallow, proprietary "200-country" tracker, and it is **not** a scrape.
It is a focused, **depth-first** reference whose value is the **judgment in the data** — sourcing and
ambiguity resolution made transparent (see [`METHODOLOGY.md`](METHODOLOGY.md)) — and whose
differentiator is the **corridor layer**: what actually clears, and what breaks, at each regulatory
boundary along a cross-border flow (see [the corridor file](hk-br-usd-stablecoin-settlement.yaml)).

Each record is one verified `(jurisdiction × instrument × dimension)` fact: structured, sourced to a
primary instrument with a pinpoint, dated, version-stamped, and machine-readable against a published
[schema](record.schema.json).

## Use
```bash
pip install -r requirements.txt
python build.py        # validates records, compiles dataset.json, regenerates coverage
```
- Machine-readable dataset: [`dataset.json`](dataset.json)
- Schema (the standard): [`record.schema.json`](record.schema.json) · vocabulary: [`taxonomy.md`](taxonomy.md)
- Add a record: copy [`_TEMPLATE.yaml`](_TEMPLATE.yaml); see [`CONTRIBUTING.md`](CONTRIBUTING.md)

**Explore it.** [`index.html`](index.html) is a self-contained static site with two views — an
interactive coverage **matrix** and a sortable, multi-filter **table** (filter by jurisdiction,
dimension, status, confidence, or keyword; click any cell or row to inspect its source and
pinpoint). Deploys as-is to GitHub Pages or Netlify (no build step; publish the repo root).

**Query it from an agent.** [`mcp_server.py`](mcp_server.py) wraps the dataset in typed query
tools over the Model Context Protocol (`query`, `compare_dimension`, `jurisdiction_profile`,
`search`, `coverage`, …). See [`MCP_SERVER.md`](MCP_SERVER.md).

## Name
`Cross-Border Stablecoin Register` (CBSR). The name foregrounds the differentiator — a
cross-jurisdictional comparison, not a single-jurisdiction tracker — and reads as a standing
reference object rather than a payments product. The citable anchor is the **DOI**, not a domain;
a custom domain (a `.org`) is optional polish. The name appears in this README, `CITATION.cff`,
the dataset, and the landing page.
